# 全球态势感知与挖掘平台 (GSAM) 终极功能规格说明书

## 1. 系统元信息
- **系统名称**：GSAM (Global Situational Awareness & Mining)
- **核心框架**：React 18 + Cesium.js + Tailwind CSS
- **设计风格**：深色拟态 (Dark-Glassmorphism) / 战术指挥风格
- **适配环境**：支持 3D 加速渲染的 Web 浏览器

---

## 2. 态势综合 (Situational Integration) 模块

### 2.1 地图可视化控制
- **渲染引擎**：Cesium 3D Globe。
- **坐标系统**：WGS84 椭球体坐标，支持海拔高度可视化。
- **视角管理**：
  - **飞行过渡 (FlyTo)**：切换区域（如亚太至中东）时，相机执行平滑插值飞行，动画时长 1.5s。
  - **视角模式**：支持 3D 球体视角与 2D 地图视角（基于 Cesium Scene Mode 切换）。
- **四维图层开关**：
  - **航空图层**：呈现红色高亮，高度层（Altitude）实时显示。
  - **海面图层**：呈现青蓝色高亮，支持航速（Knots）展示。
  - **军事基地**：按等级（Level 1/2）分层分色渲染。
  - **冲突区域**：动态实体标记，具备脉冲动效。

### 2.3 目标搜索与交互
- **全局搜索**：支持对 100+ 模拟动态目标的名称、ID 进行即时过滤。搜索结果在球体上高亮。
- **实体点选**：
  - 支持 `LEFT_CLICK` 拾取，拾取后自动弹出底部详情浮窗。
  - **拾取逻辑**：优先拾取实体 ID 为 `target-`、`base-` 或 `conflict-` 的物体。

### 2.4 历史轨迹平滑系统 (Interpolation Logic)
- **数据获取**：按时段（24h/7d/30d）通过 `/api/track` 接口拉取历史坐标集。
- **平滑算法 (LERP)**：
  - **逻辑**：在 UI 循环中根据 `playbackIndex`（浮点数）计算当前位置。
  - **公式**：`Position = P[n] + (P[n+1] - P[n]) * (playbackIndex - n)`。
  - **性能优化**：使用 `requestAnimationFrame` 确保 60FPS 运动帧率，避免 Cesium 默认 Primitive 更新的抖动感。
- **视觉反馈**：
  - **历史背景线**：Dash 虚线（不透明度 0.2），显示目标活动的完整范围。
  - **轨迹流光线**：Glow 材质实体线，实时动态增长，跟随目标坐标变动。

## 3. 态势挖掘 (Situational Mining) 模块

### 3.1 视图逻辑
- **全页面覆盖**：采用高度磨砂玻璃质感面板，支持左中右三栏结构。
- **数据过滤器**：支持“航空、舰船、基地、冲突”四个子集的独立切换与交集搜索。

### 3.2 垂直情报板块
- **实时飞行/舰船看板**：高密度信息卡片，展示航速、偏航角、坐标、型号、战略任务简述。
- **基地扫描**：展示基地驻扎规模（Strategic/Regular）及地缘政治价值。
- **冲突深度分析**：展示冲突简报（Brief）及严重程度。

### 3.3 冲突脉络 (Event Timeline)
- **模态框触发**：点击挖掘页面的“查看脉络”进入事件还原。
- **功能点**：
  - 垂直时间轴渲染冲突起始至今的所有重大事件。
  - 静态坐标点（LatLng）标注。
  - **视觉设计**：红色发光节点，强调冲突的严重性。

---

## 4. 智能助手 (AI Intelligence Assistant)

### 4.1 核心功能
- **情报检索**：基于用户输入关键词（如“红海”、“轨迹”）触发模拟专家库回复。
- **交互细节**：
  - 支持消息历史滚动追踪。
  - 预设引导词（Hints）：一键发起深度分析指令。
  - 键盘快捷键：支持 Enter 键发送消息。

---

## 5. 异常保护与稳定性 (Technical Specs)
- **黑屏预防机制**：针对 `Cesium.Cartesian3.fromDegrees` 进行入参校验，若经纬度为 NaN 或轨迹点少于两个，则跳过渲染循环，解决点击历史轨迹可能出现的黑屏问题。
- **边界保护**：
  - 播放索引 `playbackIndex` 强制钳制在 `[0, length - 1]` 范围内。
  - `Math.floor` 索引映射：确保在渲染时间戳和进度条时，索引越界不会导致报错。
- **网络韧性**：模拟数据生成器具备随机扰动算法，确保护航轨迹看起来具有真实的不确定性（基于 `Math.random()` 偏差）。

---

## 6. 用户体验细节 (UX Specs)
- **加载态**：API 请求期间全屏展示扫描雷达动画。
- **自适应布局**：自动适配宽屏显示器，挖掘页面采用 Bento Grid 栅格布局。
- **响应式控制栏**：底部控制栏在播放模式下自动展开，非播放模式下收缩，减少视觉干扰。
